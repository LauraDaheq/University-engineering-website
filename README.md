# 🎓 University Engineering Website

A responsive frontend website inspired by the design of a university engineering faculty portal.

This project was developed to practice modern frontend web development using HTML, CSS, and JavaScript, focusing on responsive layouts, navigation, and user interface design.

> **Note:** This is an educational project inspired by the structure of a university website. It is not an official website and is not affiliated with any institution.

---

## 📖 Overview

The project recreates the experience of browsing a modern university engineering website while demonstrating frontend development skills and responsive design techniques.

---

## ✨ Features

- Responsive design
- Navigation menu
- Hero banner
- News section
- Academic programs section
- Events section
- Contact section
- Footer
- Mobile-friendly layout
